🔧 Setup Instructions

Quick Start (XSS Demo Only)

The XSS demo works immediately without any setup. Simply click "Launch Demo" above!

Full Setup (All Demos)

To run all demos with full functionality, you'll need:

1. Install Node.js

Download and install Node.js from https://nodejs.org/

2. Install MySQL (Optional)

For SQL injection demo functionality, install MySQL or MariaDB.

3. Run the Server

After installing Node.js, run these commands:

# Install dependencies
npm install

# Start the server
node server.js

# Or use the PowerShell script (Windows)
.\start-server.ps1

4. Access the Demos

Once the server is running, visit:

  • XSS Demo: http://localhost:3000/
  • Health Check: http://localhost:3000/health
  • Security Info: http://localhost:3000/api/security-info
